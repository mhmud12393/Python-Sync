# Sync 2


  Commands(CMD): 
  python Sync.py -s C:....\source -r C:....\replica -i 10  -l C:....\Log_file.txt
    
  -s = Source Folder
  
  -r = Replica Folder
  
  -i = Interval
  
  -l = Log File
-------------------
A program that syncs 2 folders and replicates all changes made whilst logginh all modifications.
